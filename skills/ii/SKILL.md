# 新项目初始化流程

按以下顺序执行，每步完成后再进行下一步。

## 第一步：运行标准初始化

运行 `/init`， 生成项目根目录 `CLAUDE.md`， 在其顶部追加以下内容：

```md
## 语言规范

- 所有对话、文档、代码注释使用简体中文
- 禁止在解释中夹杂英文，仅保留代码语法英文
- 技术术语首次出现需提供中文释义
```

## 第二步：分析项目架构，输出 docs/ARCHITECTURE.md

在这个步骤，你成为一名系统架构师，分析项目整体架构，并创建 `docs/` 目录（如不存在），生成 `docs/ARCHITECTURE.md`。

## 第三步：分析各模块，输出到 docs/modules/

为第二步中识别的每个核心模块，在 `docs/modules/` 下单独创建一个 Markdown 文件，文件名使用模块名（如 `docs/modules/auth.md`）

## 完成
所有文档生成后，输出一个摘要，列出已创建的文件清单。
